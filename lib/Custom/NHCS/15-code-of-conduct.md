# Code of Conduct

All members represent NHCS in public and online.

## Expectations
- Treat witnesses with respect.
- Avoid harassment, discrimination, or intimidation.
- Do not fabricate or exaggerate evidence.
- Protect sensitive site locations.

## Conflict Rules
Disputes are handled privately through Team Leads, then Deputy Director if unresolved.

## Disciplinary Path
1. Verbal warning
2. Written warning
3. Suspension
4. Removal vote by leadership panel

Ethics concerns can be reported directly to **Mara Quill**.
