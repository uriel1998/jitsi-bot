# Communications Protocol

## Radio Format
Use: "Call sign - Location - Status - Need." Example: "Pine-2, Ridge East, stable, request pickup."

## Channel Rules
- Channel 1: mission command
- Channel 2: logistics
- Channel 3: emergency only

## Public Messaging
Only **Petra Snow** or **Mara Quill** can release official statements.
Members may share personal experiences only after evidence review closes.

## Data Security
Do not post coordinates, witness names, or unverified footage on social media.
